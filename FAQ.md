# ❓ Frequently Asked Questions

## General Questions

### Is Birthday Studio really free?
Yes! 100% free forever. No hidden costs, no subscriptions, no premium features. Everything is completely free.

### Do I need to create an account?
No account needed. Just open the editor and start creating. No sign-up, no login required.

### Do I need coding knowledge?
Absolutely not! Birthday Studio is designed for everyone. If you can upload a photo, you can create a birthday website.

### Can I use this commercially?
Yes, you can create birthday websites for clients, friends, or anyone. It's completely free for personal and commercial use.

## Technical Questions

### Where are my files stored?
All files are embedded directly in the generated website using base64 encoding. Nothing is uploaded to any server. Everything stays in your browser.

### Will my data be private?
Yes! Since nothing is uploaded to servers, your photos, music, and messages remain completely private.

### What file formats are supported?

**Photos:**
- JPG/JPEG ✅
- PNG ✅
- GIF ✅
- WebP ✅

**Music:**
- MP3 ✅ (recommended)
- WAV ✅
- OGG ✅
- M4A ✅

### What are the file size limits?
- **Photos:** 2MB per image recommended
- **Music:** 10MB per file recommended
- **Total:** No hard limit, but larger files = slower loading

### What browsers are supported?
- Chrome ✅
- Firefox ✅
- Safari ✅
- Edge ✅
- Opera ✅

All modern browsers work perfectly!

## Usage Questions

### Can I edit after generating?
You'll need to create a new website with your changes. Keep the editor tab open to make quick adjustments and regenerate.

### Will the generated link expire?
The generated website runs in your browser. As long as you keep the window open or bookmark it, it won't expire. You can also save the page (Ctrl+S) for offline access.

### Can I download the website?
Yes! Use your browser's "Save Page As" feature (Ctrl+S) to save the complete website to your computer.

### How do I share the website?
1. Generate the website
2. Bookmark the page or copy the URL
3. Share the link with the birthday person
4. They can access it anytime

### Can I use this on mobile?
The **editor** works best on desktop/laptop for uploading files. The **generated website** works perfectly on all devices including mobile and tablets.

### How many photos can I upload?
No hard limit, but 5-10 photos work best for the gallery. Too many photos may slow down the website.

### How many friend messages can I add?
As many as you want! Each friend gets their own card in the gift page.

### Can I change the theme after generating?
You'll need to go back to the editor, select a different theme, and regenerate the website.

## Feature Questions

### How does the countdown work?
The countdown calculates the time difference between now and your set birthday date/time. It updates every second automatically.

### What happens when countdown reaches zero?
1. Music changes to "Birthday Moment" track
2. Birthday message appears
3. Confetti animation starts
4. Gift button becomes available

### How does the 3-stage music work?
- **Before:** Plays during countdown
- **On Time:** Plays when birthday arrives (30 seconds)
- **After:** Plays on gift page

Music transitions automatically based on the countdown.

### What are the 6 gift stages?
1. Meme Explosion - Fun images pop up
2. Friends' Messages - Birthday wishes
3. Memory Gallery - Photo slideshow
4. Poem - Animated poem
5. Personal Letter - Sealed letter with typewriter effect
6. Final Celebration - Grand finale

### How does the guestbook work?
Friends can leave messages on the countdown page. Messages are stored in browser localStorage and appear in the gift page Stage 2.

### Can guests see other messages?
Yes, all guestbook messages appear together in Stage 2 of the gift page.

## Troubleshooting Questions

### Music not playing?
**Solutions:**
- Click anywhere on the page (browsers block autoplay)
- Check if file format is supported (use MP3)
- Reduce file size if too large
- Try a different browser

### Photos not showing?
**Solutions:**
- Check image format (use JPG or PNG)
- Reduce file size (under 2MB)
- Upload fewer images at once
- Clear browser cache and try again

### Website looks broken?
**Solutions:**
- Refresh the page (F5)
- Clear browser cache (Ctrl+Shift+Delete)
- Try a different browser
- Regenerate the website

### Can't remove uploaded files?
**Solutions:**
- Hover over the image thumbnail
- Click the × button that appears in top-right
- If stuck, refresh the editor page

### Preview window is blank?
**Solutions:**
- Check if pop-ups are blocked
- Allow pop-ups for the site
- Try clicking the button again
- Use a different browser

### Generated website won't open?
**Solutions:**
- Check if pop-ups are blocked
- Allow pop-ups for the site
- Try "Phone Preview" first
- Check browser console for errors

## Performance Questions

### Why is my website slow?
**Common causes:**
- Large file sizes (compress your files)
- Too many photos (use 5-10 max)
- Slow internet connection
- Old browser version

### How can I make it faster?
**Tips:**
- Compress photos before uploading
- Use MP3 format for music
- Reduce number of photos
- Use smaller file sizes
- Update your browser

### Will it work offline?
After the website loads once, it can work offline if you save it (Ctrl+S).
