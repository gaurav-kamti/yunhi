# Birthday Website Generator - How to Use

## Overview
This tool lets you create a customized birthday countdown website with all the features of the original project.

## Files
- **editor.html** - The main editor interface
- **editor.js** - The JavaScript that generates the website
- **index.html** - Original project (reference)
- **gift.html** - Original gift page (reference)

## How to Use

1. **Open editor.html** in your browser

2. **Fill in the form:**
   - Birthday Person's Name (e.g., "Tulip")
   - Birthday Date & Time (pick the exact moment)
   - Your Name (who it's from)

3. **Upload Music (3 stages):**
   - Music Before Birthday - plays during countdown
   - Music On Time - plays when birthday arrives
   - Music After - plays on the gift page

4. **Upload Photos:**
   - Select multiple photos for the memory gallery
   - These will appear in the slideshow

5. **Upload Memes/GIFs:**
   - Select fun images for Stage 1 of the gift page
   - These will pop up randomly

6. **Write Messages:**
   - Poem - A special poem for the birthday person
   - Personal Letter - A heartfelt letter

7. **Add Friend Messages:**
   - Click "+ Add Friend Message"
   - Enter friend's name and their birthday wish
   - Add as many as you want

8. **Generate Website:**
   - Click "ğŸ Generate Birthday Website"
   - A new window will open with your customized website
   - The website includes both the countdown page AND the gift page
